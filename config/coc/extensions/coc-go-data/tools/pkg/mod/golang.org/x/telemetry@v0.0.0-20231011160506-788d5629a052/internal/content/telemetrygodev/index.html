Go Telemetry

Overview

Go Telemetry is a way for Go toolchain programs to collect data about their performance and usage. Uploaded data is used to help improve the Go toolchain and related tools. Go Telemetry is not built into users' binaries. See what Go Telemetry collects.

For privacy information about this service, see telemetry.go.dev/privacy.
